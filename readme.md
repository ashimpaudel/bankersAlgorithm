# Nest's Boys Hostel Website

A responsive single-page website for Nest's Boys Hostel featuring information about rooms, amenities, testimonials, and contact details.

## Getting Started

Open `index.html` in any modern browser to explore the site locally.

```bash
# from the project root
open index.html     # macOS
xdg-open index.html # Linux
start index.html    # Windows
```

## Project Structure

```
assets/
  css/
    style.css
  js/
    main.js
```
